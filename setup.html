Interview Link Tracking Setup

Step 1: Set up the Google Sheet

  1. Create a new Google Sheet (or use an existing one).
  2. Go to Extensions > Apps Script.
  3. Delete the default code and paste the contents of google-apps-script.js.
  4. Click Deploy > New deployment.
  5. Set type to Web app. Set "Execute as" to Me and "Who has access" to Anyone.
  6. Click Deploy and authorize when prompted.
  7. Copy the web app URL (it looks like https://script.google.com/macros/s/.../exec).

Step 2: Enter your webhook URL

Paste the Google Apps Script web app URL below. This saves it in your browser so all tracking pages on this domain will use it.


Step 3: Test a tracking link

After saving the webhook, open any applicant link (e.g., /i/jenson-booth.html). It should log a row in your Google Sheet with IP, city, and region, then redirect to the Zoom scheduler.
